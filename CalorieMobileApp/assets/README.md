## Required Files:

Place your logo images here with these exact names:

### 1. **icon.png**
- Size: 1024x1024 pixels
- Format: PNG with transparency
- Purpose: Main app icon

### 2. **adaptive-icon.png**
- Size: 1024x1024 pixels
- Format: PNG with transparency
- Purpose: Android adaptive icon (foreground)

### 3. **splash.png**
- Size: 1242x2436 pixels
- Format: PNG
- Purpose: Splash screen (loading screen)

---

## Quick Start:

1. Create your logo images (use Canva, Figma, or any design tool)
2. Export as PNG files with exact names above
3. Place files in this folder
4. Rebuild APK: `eas build -p android --profile preview`
